선택된 참여자:
📖 채팅방 만들기 사용법
👥 참여자 선택
  • 검색창에 사용자 이름 입력
  • 검색 결과에서 원하는 사용자 클릭
  • 선택된 참여자 영역에 추가됨
  • 제거: 사용자 태그의 × 버튼 클릭
💬 1:1 채팅
  • 참여자 1명만 선택
  • 채팅방 제목은 자동으로 상대방 이름으로 설정
  • 제목 입력란이 자동으로 숨겨집니다
  • 기존 1:1 채팅이 있으면 해당 채팅방으로 이동
👥 그룹 채팅
  • 참여자 2명 이상 선택
  • 채팅방 제목을 반드시 입력해야 함
  • 제목은 그룹 채팅의 주제나 목적을 나타냄
  • 나중에 채팅 목록에서 쉽게 구분 가능
✅ 채팅방 생성
  • "채팅방 만들기" 버튼 클릭
  • 참여자가 없으면 경고 메시지 표시
  • 그룹 채팅인데 제목이 없으면 경고 메시지 표시
  • 생성 완료 후 자동으로 채팅방으로 이동
💡 팁: 1:1 채팅은 제목을 입력할 필요가 없으며, 상대방 이름이 자동으로 채팅방 제목이 됩니다. 그룹 채팅은 명확한 제목을 지정하면 나중에 쉽게 찾을 수 있습니다.
