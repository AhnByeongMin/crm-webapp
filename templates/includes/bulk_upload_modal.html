📊 프로모션 일괄등록
1단계: 엑셀 파일 업로드
📌 사용 방법
  1. 양식 다운로드: 아래 버튼을 클릭하여 엑셀 양식을 다운로드하세요
  2. 데이터 입력: 양식의 18행부터 실제 데이터를 입력하세요 (17행은 헤더)
    • 노란색 필드(대분류, 상품명, 채널, 프로모션명, 내용, 시작일)는 필수 입력
    • 회색 필드는 선택 입력
    • 5-6행의 예시 데이터는 참고용이며 자동으로 무시됩니다
  3. 파일 업로드: 작성한 엑셀 파일을 업로드하여 미리보기 및 수정
  4. 일괄 등록: 최종 확인 후 일괄 등록 버튼 클릭
※ 제공된 양식을 다운로드하여 작성 후 업로드해주세요
📄
엑셀 파일을 여기에 드래그하거나 클릭하여 업로드
지원 형식: .xlsx, .xls
직접 입력 모드
✏️ 직접 입력 방법
  • 행 추가: 아래 "➕ 새 행 추가" 버튼을 클릭하여 빈 행을 추가하세요
  • 데이터 입력: 각 필드에 직접 데이터를 입력하세요
    • 필수 항목: 대분류, 상품명, 채널, 프로모션명, 내용, 시작일
    • 선택 항목: 금액할인, 회차면제, 중복여부, 프로모션코드, 종료일
  • 행 삭제: 각 행의 삭제 버튼으로 불필요한 행을 제거하세요
  • 일괄 등록: 입력 완료 후 하단의 "✅ 일괄 등록" 버튼을 클릭하세요
※ 필요한 만큼 행을 추가하여 입력하세요
2단계: 미리보기 및 수정
