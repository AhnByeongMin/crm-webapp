🔍 프로모션 검색 및 필터

  • 채널별: 동일 채널끼리 그룹화

✏️ 프로모션 관리 (관리자)
  • "+ 새 프로모션": 새로운 프로모션 등록
  • "일괄등록": 엑셀 파일로 여러 프로모션을 한 번에 등록
    • 양식 다운로드 → 엑셀에 데이터 입력 (18행부터 - 17행은 헤더)
    • 필수 항목(노란색): 대분류, 상품명, 채널, 프로모션명, 내용, 시작일
    • 옵션 항목(회색): 금액할인, 회차면제, 중복여부, 프로모션코드, 종료일
    • 5-6행 예시 데이터는 참고용이며 자동으로 무시됩니다
  • "수정": 기존 프로모션 내용 수정 (다른 관리자가 수정 중이면 잠금)
  • "삭제": 프로모션 삭제
  • 수정 중에는 다른 관리자가 수정할 수 없도록 자동 잠금

🏷️ 프로모션 상태
  • 진행중: 현재 사용 가능한 프로모션
  • 종료: 기간이 만료된 프로모션
  • 무기한: 종료일 없이 계속 진행
📅 종료된 프로모션 관리
  • 기본 동작: 종료된 프로모션은 자동으로 숨겨집니다
  • 히스토리 보기: "종료된 프로모션 포함" 체크박스를 체크하면 과거 프로모션 이력까지 모두 확인 가능
  • 활용: 프로모션 개수가 많아질수록 현재 진행 중인 프로모션만 보는 것이 편리합니다
  • 무기한 프로모션: 종료일이 없는 프로모션은 항상 표시됩니다
🎨 헤더 화면 설정
  • 다크모드 토글: 헤더 우측의 ☀️/🌙 버튼을 클릭하면 라이트/다크 모드 전환
  • 글자 크기 조절: 헤더의 [가][가][가] 버튼으로 작게/보통/크게 조절
  • 설정은 브라우저에 자동 저장되어 다음 접속 시에도 유지됩니다
💡 Tip: 대분류를 선택하면 해당 대분류의 상품만 상품 필터에 표시됩니다. 필터를 조합하여 원하는 프로모션을 빠르게 찾을 수 있습니다! 프로모션이 많아지면 종료된 프로모션을 숨겨서 깔끔하게 관리하세요.
