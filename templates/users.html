📖 사용자 관리 사용법
👤 사용자 추가
  • "+ 사용자 추가" 버튼을 클릭합니다
  • 이름: 사용자 이름을 입력합니다 (필수)
  • 권한: 상담사 또는 관리자 선택
  • 상태: 활성/비활성 선택 (기본: 활성)
  • 팀: 팀 배정 (선택사항)
  • 초기 비밀번호: 상담사는 body123!, 관리자는 admin1234
🔍 사용자 검색 및 필터
  • 권한 필터: 상담사/관리자별 조회
  • 상태 필터: 활성/비활성 사용자 필터링
  • 팀 필터: 특정 팀 소속 사용자만 조회
  • 이름 검색: 사용자 이름으로 검색
⚙️ 사용자 관리
  • 활성화/비활성화: 사용자의 로그인 가능 여부 설정
  • 팀 변경: 사용자의 소속 팀 변경
  • 권한 변경: 상담사 ↔ 관리자 권한 변경
  • 비밀번호 초기화: 기본 비밀번호로 리셋
  • 삭제: 사용자 계정 완전 삭제 (복구 불가)
👥 팀 관리
  • "팀 관리" 버튼을 클릭합니다
  • 새 팀은 사용자 추가/수정 시 자동 생성됩니다
  • 팀 삭제 시 해당 팀원의 팀 정보가 모두 제거됩니다
🎨 헤더 화면 설정
  • 다크모드 토글: 헤더 우측의 ☀️/🌙 버튼을 클릭하면 라이트/다크 모드 전환
  • 글자 크기 조절: 헤더의 [가][가][가] 버튼으로 작게/보통/크게 조절
  • 설정은 브라우저에 자동 저장되어 다음 접속 시에도 유지됩니다
💡 Tip: 비활성화된 사용자는 로그인할 수 없지만 데이터는 보존됩니다. 일시적으로 접근을 차단하려면 삭제 대신 비활성화를 사용하세요!
