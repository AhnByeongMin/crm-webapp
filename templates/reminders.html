📖 내 예약 관리 사용법
📅 캘린더 사용하기
  • 날짜 클릭: 해당 날짜의 예약만 표시
  • 다시 클릭: 필터 해제 (전체 예약 표시)
  • 노란색 표시: 예약이 있는 날짜
  • 빨간색 표시: 오늘 예약이 있는 날짜
  • 오늘 버튼: 현재 월로 이동
📊 통계 정보
  • 전체: 모든 예약 개수
  • 미완료: 아직 완료되지 않은 예약
  • 완료: 완료 처리된 예약
  • 통계는 필터와 무관하게 항상 전체 데이터 기준입니다
📝 예약 추가하기
  • "+ 새 예약 추가" 버튼 클릭
  • 캘린더에서 날짜를 선택한 상태면 자동으로 날짜 입력됨
  • 개인정보 보호: 전화번호 입력 시 저장이 차단됩니다
🔍 검색 및 필터
  • 검색창: 제목이나 내용으로 검색
  • 완료된 항목 보기: 체크하면 완료된 예약도 표시
  • 캘린더 날짜 선택: 특정 날짜의 예약만 표시
🔔 알림 설정
  • 알림 설정 버튼: 캘린더 하단의 버튼 클릭
  • 예약 알림 시간: 예약 몇 분 전에 알림 받을지 설정 (5분~1시간)
  • 반복 알림: 미완료 예약에 대해 반복 알림 (간격, 종료 시점 설정)
  • 아침 일일 요약: 매일 설정된 시간에 오늘 예약 목록 알림
  • 테스트 버튼: 일일 요약 알림 즉시 발송 테스트
💡 Tip: 알림을 받으려면 브라우저에서 알림 권한을 허용해야 합니다!
🎨 헤더 화면 설정
  • 다크모드 토글: 헤더 우측의 ☀️/🌙 버튼을 클릭하면 라이트/다크 모드 전환
  • 글자 크기 조절: 헤더의 [가][가][가] 버튼으로 작게/보통/크게 조절
  • 설정은 브라우저에 자동 저장되어 다음 접속 시에도 유지됩니다
💡 개인정보 보호: 전화번호 대신 상담번호(CO), 고객번호(CU), 또는 이름을 사용해주세요.
