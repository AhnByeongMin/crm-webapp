📖 채팅방 사용법
💬 메시지 보내기
  • 텍스트 입력: 하단 입력창에 메시지 작성
  • Enter: 메시지 전송
  • Shift + Enter: 줄바꿈
  • 이모지 버튼 (😊): 이모지 선택 및 전송
  • 파일 첨부 (📎): 이미지, 문서 등 파일 첨부
📎 파일 공유
  • 파일 첨부: 📎 버튼 클릭 또는 드래그 앤 드롭
  • 이미지: 채팅창에서 바로 미리보기, 클릭하면 크게 보기
  • 문서 파일: 클릭하여 다운로드
  • 여러 파일: 한 번에 여러 파일 첨부 가능
👁️ 읽음 표시
  • 빨간 숫자: 아직 읽지 않은 사람 수 표시
  • 0명: 모든 참여자가 읽음
  • 자동 업데이트: 상대방이 읽으면 실시간 반영
  • 채팅방 입장: 자동으로 읽음 처리됨
⌨️ 입력 중 표시
  • 상대방이 입력 중: "OOO님이 입력 중..." 표시
  • 실시간 반영: 상대방이 입력을 시작하면 즉시 표시
  • 여러 명 입력: 여러 사람이 동시에 입력 중일 때 모두 표시
🔔 알림 기능
  • 브라우저 알림: 채팅방 밖에서도 새 메시지 알림
  • 읽지 않은 메시지: 채팅 목록에서 빨간 배지로 표시
  • 당일 예약 배너: 오늘 예약이 있으면 상단 배너 표시
  • 알림 권한: 브라우저에서 알림 권한 허용 필요
🎨 헤더 화면 설정
  • 다크모드 토글: 헤더 우측의 ☀️/🌙 버튼을 클릭하면 라이트/다크 모드 전환
  • 글자 크기 조절: 헤더의 [가][가][가] 버튼으로 작게/보통/크게 조절
  • 설정은 브라우저에 자동 저장되어 다음 접속 시에도 유지됩니다
💡 팁: 이미지를 클릭하면 크게 볼 수 있으며, 파일명을 클릭하면 다운로드됩니다. Shift + Enter로 여러 줄 메시지를 작성할 수 있습니다.
