## 2. 공통 요소

### 2.1 헤더 네비게이션

```
┌─────────────────────────────────────────────────────────────────────┐
│ 📋 페이지 제목                    [할일] [채팅 🔴3] [내 예약] [마이페이지] [로그아웃] │
└─────────────────────────────────────────────────────────────────────┘
```

| 요소 | 설명 |
|------|------|
| 페이지 제목 | 현재 페이지명 표시 |
| 할일 | 할일 목록 이동 (배지: 미완료 건수) |
| 채팅 | 채팅 목록 이동 (배지: 안읽은 메시지) |
| 내 예약 | 예약 목록 이동 (배지: 오늘 예약 건수) |
| 프로모션 게시판 | 프로모션 목록 이동 |
| 마이페이지 | 개인 설정 이동 |
| 로그아웃 | 세션 종료 |

### 2.2 알림 모달

```
┌─────────────────────────────────┐
│ 🔔 알림                    [X] │
├─────────────────────────────────┤
│                                 │
│        알림 메시지 내용          │
│                                 │
├─────────────────────────────────┤
│                      [확인]     │
└─────────────────────────────────┘
```

| 타입 | 헤더 색상 | 아이콘 |
|------|-----------|--------|
| info | 보라색 | 💬 |
| success | 녹색 | ✅ |
| warning | 노란색 | ⚠️ |
| error | 빨간색 | ❌ |
